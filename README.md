# Przyjemny, intuicyjny kreator

Możesz swobodnie i prosto stworzyć grafikę pikselową jak z dawnych czasów, a następnie zapisać ją na swoim komputerze.

# Użyte technologie:

- HTML,
- CSS/SASS,
- JavaScript,
- html2canvas,
- html2image.
